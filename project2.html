Analysis of Hashing Algorithms on FPGAs

Project Overview

This project compares the performance and efficiency of MD5 and MD6 hashing algorithm implementations across different programming languages: Verilog, C++, and OpenCL. The focus is on evaluating execution time and leveraging parallelisation capabilities to enhance performance on FPGAs.

Objectives

  • Implement MD5 and MD6 hashing algorithms in Verilog, C++, and OpenCL.
  • Evaluate the performance of each implementation.
  • Compare the efficiency of software and hardware implementations.
  • Explore the potential of parallelisation in improving performance.

Technologies Used

  • MD5 and MD6 hashing algorithms
  • Verilog for FPGA implementation
  • C++ for software implementation
  • OpenCL for parallel computing
  • Nexys A7 FPGA development board

Project Details

The project involves the following tasks:

  1. Sequential implementation of MD5 in C++ to establish a performance baseline.
  2. Hardware implementation of MD5 using Verilog on an FPGA.
  3. Parallel implementation of MD5 using OpenCL to explore parallelisation capabilities.
  4. Sequential and parallel implementations of MD6 in C++.
  5. Performance evaluation based on execution time and throughput.

Results

The analysis showed that the parallelised MD6 implementation in C++ provided the best performance across various input sizes. The hardware implementation of MD5 using Verilog demonstrated significant speedup for small input sizes, highlighting the efficiency of dedicated hardware for specific tasks.
