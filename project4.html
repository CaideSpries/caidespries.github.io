Hardware Implementation of Online Filters

Project Overview

This project investigates the implementation of three different types of filters on an ESP32 Cam board to determine which performs best on floating point hardware: Infinite Impulse Response (IIR) filters, Finite Impulse Response (FIR) filters, and Discrete Fourier Transform (DFT) filters. The study aims to enhance signal quality and data accuracy in various real-time applications.

Objectives

  • Investigate literature on different filter implementation methods.
  • Implement FIR, IIR, and DFT filters on an ESP32 Cam board.
  • Evaluate the performance of each filter type in real-time applications.
  • Analyse the computational complexity and efficiency of each filter.
  • Determine the optimal filter for various real-time processing applications.

Technologies Used

  • ESP32 Cam board
  • MATLAB for filter design
  • Arduino IDE for implementation
  • Python for simulation and testing
  • Various digital filter design techniques (FIR, IIR, DFT)

Project Details

The project involves the following tasks:

  1. Designing FIR, IIR, and DFT filters using MATLAB and implementing them on an ESP32 Cam board.
  2. Testing the filters with real-time signal inputs to evaluate their performance.
  3. Measuring and comparing the computational efficiency and accuracy of each filter type.
  4. Analysing the results to determine the most suitable filter for specific real-time applications.

Results

The project concluded that the IIR filter performs best in terms of time delay and computational efficiency, making it ideal for real-time filtering applications. The FIR filter showed a trade-off between time delay and filter accuracy, while the DFT filter was found to be the least suitable for real-time applications due to its high computational complexity and poor performance.
