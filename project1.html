Mobile and Wireless Networks Project

Project Overview

This project focuses on optimising vertical handoff decisions in heterogeneous networks (HetNets) using Multi-Attribute Decision Making (MADM) techniques. The study explores various algorithms to enhance connectivity and user experience in mobile and wireless networks.

Objectives

  • Investigate literature on vertical handoff decision algorithms for HetNets.
  • Specify technologies and criteria for network selection algorithms using TOPSIS.
  • Develop a diagram of the heterogeneous network.
  • Create flowcharts for RAT-selection procedures.
  • Formulate equations for normalizing decision matrices and ranking RATs.
  • Test various criteria on RAT selection.

Technologies Used

  • Wi-Fi (WLAN)
  • 2G (GSM/EDGE)
  • 4G (LTE)
  • MADM Techniques (TOPSIS)
  • Python for simulation and code setup

Project Details

The project encompasses the following tasks:

  1. Literature Review: A comprehensive review of existing vertical handoff decision algorithms and MADM techniques.
  2. Specification of Technologies and Criteria: Identifying RATs and establishing selection criteria and decision matrices for network selection algorithms.
  3. Network Diagram: Creating a diagram to visualize the heterogeneous network setup.
  4. Flowcharts: Developing flowcharts to illustrate the procedures for making RAT-selection decisions.
  5. Equations: Formulating equations to normalize decision matrices and rank RATs for new and vertical handoff scenarios.
  6. Simulation and Testing: Setting up and running Python simulations to test the effect of various criteria on RAT selection.

Results

The simulations demonstrated the effectiveness of different MADM techniques in optimising vertical handoff decisions. Wi-Fi emerged as the preferred RAT in most scenarios due to its superior bandwidth and lower delay and power consumption values. The project underscored the importance of considering multiple criteria in RAT selection to enhance user experience and network performance.
