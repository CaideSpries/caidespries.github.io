Mobile and Wireless Networks Project

Project Overview

In today's connected world, mobile devices often have access to multiple types of wireless networks simultaneously, such as Wi-Fi, 4G, and 2G. These are known as Heterogeneous Networks (HetNets). The challenge is to enable a device to seamlessly switch between these networks to maintain the best possible connection quality. This project investigates how to optimise these switching decisions, known as 'vertical handoffs.' The core of this project was to develop and simulate a network selection algorithm using the TOPSIS (Technique for Order Preference by Similarity to Ideal Solution) method to intelligently choose the best network based on multiple criteria.

Technologies Used

  • Wi-Fi (WLAN)
  • 2G (GSM/EDGE)
  • 4G (LTE)
  • MADM Techniques (TOPSIS)
  • Python for simulation and code setup

Methodology and Process

The project followed a structured approach to solve the network selection problem. The core of the methodology was the TOPSIS algorithm, which provides a logical framework for ranking the available networks. The process, illustrated in the flowchart below, involves several key steps:

  • Defining the Criteria: Establishing the metrics for comparison (Allowed Bandwidth, Delay, Power Consumption).
  • Building the Decision Matrix: Assigning performance values to each network for every criterion.
  • Normalizing and Weighting: Processing the data to allow for fair comparison and applying user-defined preferences (weights).
  • Calculating Closeness: Determining which network is closest to the 'ideal' solution.
  • Selecting the Best Network: Choosing the RAT with the highest closeness coefficient.

Conclusion

This project successfully demonstrated the effectiveness and complexities of using the TOPSIS algorithm for vertical handoff decisions. The initial simulations, based on realistic data, confirmed that the algorithm correctly identifies an objectively superior network. However, the key insight came from the analysis with a balanced decision matrix. It revealed that the user's specific prioritiesâ€”whether for speed, responsiveness, or battery lifeâ€”are a critical factor that can dynamically change the optimal network choice. The results show how changing the decision matrix significantly impacts network selection, and when combined with user preferences, highlights the complexities that warrant further research. Ultimately, the project proves that effective network selection is not just about the raw performance of technologies, but about the intricate interplay between network capabilities and user needs.
