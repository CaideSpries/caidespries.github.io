Camera Trap Monitoring System

Project Overview

This project focuses on improving data storage and transmission for camera traps used in monitoring wildlife, specifically focusing on Marshall Eagles in the Kruger National Park. It addresses challenges such as data loss, power efficiency, and the need for continuous monitoring in harsh environmental conditions.

Objectives

  • Investigate current methods and hardware used for monitoring bird nests and feeding habits.
  • Develop a comprehensive system for capturing, storing, and transmitting data from camera traps.
  • Incorporate temperature monitoring to provide additional data for research.
  • Ensure the system is cost-effective and reliable under harsh environmental conditions.

Technologies Used

  • ESP32CAM board with OV2640 camera
  • PIR motion detection sensor
  • DHT11 temperature and humidity sensor
  • MicroSD card for data storage
  • Raspberry Pi for base station data management
  • Python for data processing and system integration

Project Details

The system design includes capturing high-resolution images at regular intervals, detecting motion to reduce 'dead' periods, and storing data on a microSD card with timestamps. The temperature and humidity data are also recorded and stored. A wireless transmission system ensures that data can be collected without the need to climb trees, enhancing the safety and efficiency of the research process.

  1. Design and integrate a low-power image capturing and storage system.
  2. Implement a motion detection system to optimise data collection periods.
  3. Incorporate a temperature and humidity monitoring system.
  4. Ensure reliable wireless data transmission to a base station.
  5. Validate the system through rigorous testing and refinement.

Results

The redesigned camera trap system successfully addressed the client's challenges, providing a reliable and efficient method for monitoring raptor nests. The system's ability to store and transmit data wirelessly minimised the need for physical retrieval, reducing the risk of data loss and improving overall research efficiency.
